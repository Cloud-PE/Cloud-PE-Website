# How to load CE plugins
## Automatic loading
Put `Cloud-PE.iso` onto a Ventoy drive, create a `ce-apps` folder on that Ventoy drive, and place your CE plugins inside it — they will load automatically when entering PE.

![](https://pic.imgdb.cn/item/66e7b942d9c307b7e9dc92c4.png)
## Manual loading
1. In Cloud-PE, double-click a plugin file to load it

2. Load via the command line (using Cloud Assistant):

````cp-help.exe <plugin path (quote it if it contains spaces)>````
