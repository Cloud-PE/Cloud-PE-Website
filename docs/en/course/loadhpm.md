# How to use HotPE modules
We recommend [CE plugins](https://files.cloud-pe.cn/%E4%B8%8B%E8%BD%BD/CE-%E6%8F%92%E4%BB%B6).

Cloud-PE has supported HotPE modules since its very first release.
## How to load HotPE modules?
### Automatic loading
1. Same as the [HotPE method for loading HotPE modules](https://docs.hotpe.top/course/loadhpm.html)

2. Put `Cloud-PE.iso` onto a Ventoy drive, create a `HotPEModule` folder on that drive, and place your HotPE modules inside it — they will load automatically when entering PE.
### Manual loading
1. In Cloud-PE, double-click a HotPE module file to load it

2. Load via the command line (using CE Assistant):

````cp-help.exe <plugin path (quote it if it contains spaces)>````
## Downloading HotPE modules inside PE
Cloud-PE uses the Cloud Plugin Market (a third-party HotPE module store) built by the Cloud-PE team to obtain HotPE modules. On the Cloud-PE desktop, open the Cloud Plugin Market, switch to a HotPE source, and download.
